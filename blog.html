What is document object model?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from DOM

We can use various methods to select an element such as getElementById . getElementById method allows us to select an element by its id. Using getElementsByClassName, Using getElementsByTagName, Using getElementsByTagName and Using querySelectorAll we can select an element from HTML DOM .

What is the event delegation in the context of DOM

Even delegation means attach a listener to a common container . Instead of attaching individual event listeners to each child element, we attach a single event listener to a common container element higher in the DOM.
